# 3D-Printer-
A 3D printer is an additive manufacturing device that builds three-dimensional objects by depositing materials (like plastic, resin, or metal) one thin layer at a time, following a computer-designed model (STL file).

# 3D Printer

A high-quality desktop 3D printer for rapid prototyping, hobby projects, and small-scale manufacturing.

## Features

- **Build Volume**: 220 × 220 × 250 mm
- **Printing Technology**: FDM (Fused Deposition Modeling)
- **Layer Resolution**: 0.05 - 0.3 mm
- **Print Speed**: Up to 250 mm/s
- **Filament Support**: PLA, PETG, ABS, TPU, ASA
- **Heated Bed**: Yes (up to 110°C)
- **Hotend**: 300°C capable
- **Auto Bed Leveling** (optional)
- **WiFi & Touchscreen** support

## What's Included

- 3D Printer assembled & calibrated
- Power supply & cable
- 1kg PLA filament
- Tool kit & spare parts
- USB & SD card
- User manual

## Quick Start

1. Unbox and place on a flat surface
2. Insert filament
3. Level the bed (auto or manual)
4. Slice your model using Cura / PrusaSlicer / Bambu Studio
5. Start printing!
